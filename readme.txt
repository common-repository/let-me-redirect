The magic begins as it spontaneously redirect a user to a page when viewing a post/page on your website for more than a given limit.

== Description ==
LET ME REDIRECT permits site administrator to re-direct a non-logged in user by following a picked measure of seconds when seeing a page or post on the site. The module directions are exceptionally straightforward. Add a shortcode to the post or a page to redirect the user.

= Features =
* Its Magic works on pages or posts.
* NO major mess up with some settings or configurations. Just a simple shortcode.
* Need a URL to redirect the user.
* Time in seconds to wait before re-directing the user.
* One plugin you need for redirect to various pages.

== Installation ==
1. Upload 'let-me-redirect.zip' to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Add the shortcode on the pages or posts that you want to redirect. Shortcodes should look like [REDIRECT_ME url='http://anysiteyouwanttoredirect.com/' sec='10'] where the url= part is the URL to redirect to, and sec= is the seconds to wait before redirecting the page.

4. Thats all folks!

== Note ==
 LET ME REDIRECT should work with older versions of WordPress as well but was not tested with anything older than 3.0
